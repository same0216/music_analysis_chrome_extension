# インストール方法

## 1. アイコンの生成

拡張機能を使用する前に、アイコンファイルを生成する必要があります。

### 方法A: HTMLファイルを使用（簡単）

1. `generate-icons.html` をブラウザで開く
2. 「Download All」ボタンをクリック
3. ダウンロードされた3つのPNGファイル（icon16.png, icon48.png, icon128.png）を `icons/` フォルダに移動

### 方法B: 自分でアイコンを作成

`icons/` フォルダに以下のサイズのPNGファイルを配置してください：
- icon16.png (16x16ピクセル)
- icon48.png (48x48ピクセル)
- icon128.png (128x128ピクセル)

音楽ノートやウェーブフォームのデザインがおすすめです。

## 2. Chrome拡張機能のインストール

1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このプロジェクトのフォルダを選択
6. 拡張機能がツールバーに表示されます

## 3. 使用方法

1. 音楽が再生されているウェブページ（YouTube、Spotify Web Player、SoundCloudなど）を開く
2. ツールバーの拡張機能アイコンをクリック
3. 「Start Analysis」ボタンをクリック
4. 数秒待つと、BPMとキーが表示されます

## トラブルシューティング

### 「Could not capture audio」エラーが出る場合
- タブで音楽が再生されていることを確認してください
- ページをリロードして再度試してください
- ブラウザで音声が有効になっていることを確認してください

### アイコンが表示されない場合
- `icons/` フォルダに3つのPNGファイルがあることを確認してください
- ファイル名が正確に一致していることを確認してください（icon16.png, icon48.png, icon128.png）

### 分析が正しく動作しない場合
- 音楽が一定のビートを持っていることを確認してください
- 音量が十分であることを確認してください
- 別の音楽で試してみてください

## 対応サイト

- Spotify Web Player
- SoundCloud
- Apple Music Web
- その他、HTML5オーディオを使用するサイト
